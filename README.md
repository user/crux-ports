# crux-ports
Personal ports collection

### Installation

1. Add the repo
 - `# wget -O /etc/ports/ivs.git https://git.io/v2z83`
2. Enable collection in your prt-get.conf
 - Add `prtdir /usr/ports/ivs` in your /etc/prt-get.conf
3. Sync the repo
 - `# ports -u`
 - `# prt-get cache`
4. Install the package
 - `prt-get depinst`
